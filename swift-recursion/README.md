# Задача реализации быстрой сортировки

Нужно отсортировать студентов в порядке убывания их среднего балла с помощью рекурсивной быстрой сортировки.

Список студентов находится в файле `students.json`:

```json
{
    "group_1": {
      "Иван": [19, 95],
      "Мария": [20, 91],
      "Марина": [21, 60],
      "Алексей": [20, 70]
    },
    "group_2": {
      "Никита": [20, 99],
      "Анна": [21, 80],
      "Антон": [20, 90],
      "Виктор": [19, 95],
      "Вика": [21, 70]
    }
}
```
